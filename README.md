# DataSecurity Project

## Overview
This project provides a collection of cryptographic algorithms and techniques for data security. It includes implementations of well-known encryption and hashing methods such as AES, DES, RSA, MD5, and more. The goal of this project is to demonstrate various methods of securing data through encryption.

## Features
- **Encryption Algorithms:** AES, DES, RSA, and others.
- **Hashing Algorithms:** MD5 for message authentication.
- **Cryptographic Techniques:** Includes techniques like Hill Cipher, PlayFair, and others.
- **Exceptions Handling:** Ensures that errors in cryptographic operations are managed effectively.

## Algorithms Implemented
- **AES (Advanced Encryption Standard)**
- **DES (Data Encryption Standard)**
- **ElGamal Encryption**
- **RSA (Rivest-Shamir-Adleman)**
- **MD5 (Message Digest Algorithm)**
- **Autokey Vigenere Cipher**
- **Caesar Cipher**
- **Columnar Transposition Cipher**
- **Hill Cipher**
- **Monoalphabetic Cipher**
- **Playfair Cipher**
- **Rail Fence Cipher**
- **Repeating Key Vigenere Cipher**
